## Interaction 1: Understanding CSS Grid Layout

**AI Tool Used:** ChatGPT

**Date:** 15 July 2026

**Prompt Given:**
How can I create a website layout using CSS Grid?

**Why I Asked:**
I wanted to understand how CSS Grid works and how it can be used to arrange different sections of a webpage.

**AI Response Summary:**
The AI explained that CSS Grid is used for creating two-dimensional layouts using rows and columns. It explained properties like `display: grid`, `grid-template-columns`, `grid-template-rows`, and `gap`.

**How I Used It:**
I applied the CSS Grid concepts to structure my webpage layout and adjusted the grid values according to my design requirements.

**What I Learned:**
I learned how CSS Grid helps create organized layouts and how rows, columns, and gaps control the placement of elements.
